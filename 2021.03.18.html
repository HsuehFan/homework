getAttribute, setAttribute 應用(參考章節)

以下連結為 HTML 的屬性列表，可以透過 getAttribute, setAttribute 操作這些屬性。

參考資料

getAttribute

    1. 使用 getAttribute 可以回傳一個元素上的指定屬性值，語法如下：
    element.getAttribute("屬性名稱");
    範例：
目標 Id
    2. 使用 getAttribute 取用屬性時，如果指定的屬性不存在，則會回傳 null 或 ""。

setAttribute

    1. 可以使用 setAttribute 替指定元素設定屬性值，語法如下：
    element.getAttribute("屬性名稱", "屬性值");
